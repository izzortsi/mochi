2. If several sets $A, B, C, \ldots$ all are subsets of the same set $X$ then the differences $X \setminus A, X \setminus B, X \setminus C, \ldots$ are the **complements** of $A, B, C, \ldots$ in $X$ and are denoted $A^c, B^c, C^c, \ldots$. The symbol $A^c$ is read “$A$ complement.”

(a) Prove that $(A^c)^c = A$.

(b) Prove **De Morgan’s Law**: $(A \cap B)^c = A^c \cup B^c$ and derive from it the law $(A \cup B)^c = A^c \cap B^c$.

(c) Draw Venn diagrams to illustrate the two laws.

(d) Generalize these laws to more than two sets.
