(i.e., $\mathcal{L}[u] = du/dx + x^2u$).

Um operador é dito **linear** se verifica:
i) $\mathcal{L}[u + v] = \mathcal{L}[u] + \mathcal{L}[v]$,
ii) $\mathcal{L}[cu] = c\mathcal{L}[u]$,
para "todas" funções $u, v$ e toda constante $c$.

A equação

$$\mathcal{L}[u] = 0$$

(5)

é dita linear se $\mathcal{L}$ é um operador linear. A equação (5) é dita uma **equação linear homogênea**.

A equação

$$\mathcal{L}[u] = g$$

(6)

onde $g \neq 0$ é uma função dada da variável indipendente, é dita linear se $\mathcal{L}$ é um operador linear. A equação (6) é dita uma **equação linear não-homogênea**.

**Exemplo:** Sejam $p(t)$ e $q(t)$ duas funções continuas definidas para $a < t < b$. A equação diferencial

$$\frac{dx}{dt} + p(t)x = g(t)$$

(7)

é uma equação linear da primeira ordem. É homogênea se $g = 0$, e não-homogênea se $g \neq 0$. Para uma dada equação da forma (7), a equação

$$\frac{dx}{dt} = -p(t)x$$

(8)

é dita a sua equação homogênea **associada**.

Observe que a equação (8) com condição inicial $x(t_o) = x_o$ tem uma solução

$$x(t) = x_o e^{-\int_{t_o}^t p(\tau)d\tau}$$
