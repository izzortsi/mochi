matriz de $S$ (vista como transformação de $V^\perp$ em $V^\perp$) é diagonal superior. Assim, como, para cada $k = 2, \ldots, n$, vale

$$S e_k = a_{2k} e_2 + \ldots + a_{kk} e_k,$$

e $PT e_k = a_{1k} e_1$, segue

$$T e_k = PT e_k + S e_k = a_{1k} e_1 + a_{2k} e_2 + \ldots + a_{kk} e_k,$$

o que mostra que a matriz de $T$ na base (ortonormal) $e_1, e_2, \ldots, e_n$, que chamaremos de $M$, é triangular superior. Como o determinante de $T$ é igual ao de sua matriz em qualquer base e o determinante de matriz triangular superior é igual ao produto dos elementos da diagonal, então o polinômio característico de $T$ é $\det(xI - M) = (x - a_{11}) \cdots (x - a_{nn})$. Logo, os autovalores de $T$, que são as raízes do polinômio característico, coincidem (inclusive quanto à multiplicidade) com os elementos da diagonal de $M$.

**Escólio:** A Forma de Schur é equivalente à existência de base ortonormal de $E$, dita base de Schur, formada pelos vetores $e_1, \ldots, e_n$, e números complexos, $\lambda_1, \ldots, \lambda_n$, tais que, sendo $E_k$ o espaço gerado por $e_1, \ldots, e_k$, $k = 1, \ldots, n$, e $E_o = \{0\}$, então

$$(T e_k - \lambda_k e_k) \in E_{k-1} \forall k = 1, \ldots, n.$$

### b O Teorema de Cayley-Hamilton

Embora Cayley-Hamilton seja um teorema famoso, vamos, aqui, obtê-lo como corolário do Schur,

**Teorema de Cayley-Hamilton:** Se $A$ é matriz $n \times n$ com entrada em $\mathbb{C}$ e $p$ é o polinômio característico de $A$ (dado por $p(x) = \det(xI - A)$), então $p(A) = 0$.

Demonstração : Comecemos fixando uma base de Schur, $\alpha = \{e_1, e_2, \ldots, e_n\}$, correspondente à transformação linear $T$ dada por $A$. Se $U$ é a matriz de mudança de base de $\alpha$ para a base canônica, temos

$$p(x) = \det(xI - A) = \det(U^{-1}(xI - A)U) = \det(xI - M) = (x - \lambda_1) \cdots (x - \lambda_n).$$

Temos, também, que, sendo $E_k$ o espaço gerado por $e_1, \ldots, e_k$, $k = 1, \ldots, n$, e $E_o = \{0\}$, então

$$(T e_k - \lambda_k e_k) \in E_{k-1} \forall k = 1, \ldots, n.$$

Daí segue que

$$(T - \lambda_1 I) \cdots (T - \lambda_k I) e_k = 0 \forall k = 1, \ldots, n.$$
