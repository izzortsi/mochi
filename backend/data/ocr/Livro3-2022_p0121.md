$$\langle Tu, v \rangle = \langle u, Tv \rangle \quad \forall u, v \in E_0,$$

nossa primeira versão do Teorema Espectral, já demonstrada, para o caso de dimensão 2, assegura que $E_0$ tem base ortonormal, $\{\varepsilon_1, \varepsilon_2\}$, de autovetores de $T$. Juntando, $\varepsilon_3$, temos a base de $E$, $\{\varepsilon_1, \varepsilon_2, \varepsilon_3\}$, de autovetores de $T$. Só falta enunciar o Teorema Espectral, caso tridimensional.

**Teorema:** Se $T : R^3 \rightarrow R^3$ é transformação linear autoadjunta, então $R^3$ tem base ortonormal de autovetores de $T$.

**Exercício 18.4** Você consegue fazer o caso tetradimensional? Isto é: Suponha que $T : R^4 \rightarrow R^4$ seja transformação linear dada, na base canônica, por matriz simétrica. Mostre que $R^4$ tem base ortonormal de autovetores de $T$.
