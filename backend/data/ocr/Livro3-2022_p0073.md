$$0 = Ax^2 + Dx + Ey + F = A \left( x + \frac{D}{2A} \right)^2 + Ey + F - \left( \frac{D}{2A} \right)^2 =$$
$$= Au^2 + Ev + G,$$

com

$$u = x + \frac{D}{2A}, \quad v = y, \quad G = F - \left( \frac{D}{2A} \right)^2.$$

Obtemos, agora, os seguintes casos:

(i) $E \neq 0 \longrightarrow \text{parábola}$
(ii) $E = 0, G = 0 \longrightarrow \text{uma reta}$
(iii) $E = 0, AG > 0 \longrightarrow \text{conjunto vazio}$
(iv) $E = 0, AG < 0 \longrightarrow \text{par de retas paralelas}$
