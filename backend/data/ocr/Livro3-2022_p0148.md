cuja norma, ao quadrado, é

$$\left( \sum_{i=1}^{k} \langle u_j, w_i \rangle^2 \right) \sigma_j^2.$$

Portanto,

$$\left| \Sigma(E) \right|^2 = \sum_{j=1}^{p} \left( \sum_{i=1}^{k} \langle u_j, w_i \rangle^2 \right) \sigma_j^2 = \sum_{j=1}^{p} \alpha_j \sigma_j^2,$$

com

$$0 \leq \alpha_j = \left( \sum_{i=1}^{k} \langle u_j, w_i \rangle^2 \right) \leq |u_j|^2 = 1, \ j = 1, \ldots, p$$

e

$$\sum_{j=1}^{p} \alpha_j = \sum_{j=1}^{p} \left( \sum_{i=1}^{k} \langle u_j, w_i \rangle^2 \right) = \sum_{i=1}^{k} \left( \sum_{j=1}^{p} \langle u_j, w_i \rangle^2 \right) \leq \sum_{i=1}^{k} |w_i|^2 \leq k.$$

Diante das circunstâncias, dado que os $\sigma^2$ são decrescentes, o melhor que podemos fazer para maximizar $\left| \Sigma(E) \right|^2$ é concentrar os pesos nos $k$ primeiros valores, se possível atribuindo-lhes os maiores valores possíveis. Explicitamente, temos

$$\left| \Sigma(E) \right|^2 \leq \sum_{j=1}^{k} \sigma_j^2.$$

Ora, escolhendo $E$ gerado por $u_1, \ldots, u_k$, temos $\Sigma(E) = \Sigma_k$ coincidindo com $\Sigma$ em todas as entradas, exceto nas correspondentes aos valores singulares $\sigma_{k+1}, \ldots, \sigma_l$ (ou seja: coincidindo com $\Sigma$ nos $k$ primeiros elementos da diagonal principal e tendo nulas as demais entradas), e obtemos, exatamente,

$$\left| \Sigma_k \right|^2 = \sum_{j=1}^{k} \sigma_j^2.$$

O Teorema está demonstrado.
