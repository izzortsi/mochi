11. A random variable $\xi$ takes the values 1, 2, ..., $n$, ... with probabilities

$$\frac{1}{3}, \frac{1}{3^2}, \ldots, \frac{1}{3^n}, \ldots$$

Find $E\xi$.

12. Balls are drawn from an urn containing $w$ white balls and $b$ black balls until a white ball appears. Find the mean value $m$ and variance $\sigma^2$ of the number of black balls drawn, assuming that each ball is replaced after being drawn.

Ans. $m = \frac{b}{w}, \sigma^2 = \frac{b(w + b)}{w^2}$.

13. Find the mean and variance of the random variable $\xi$ with probability density

$$p_\xi(x) = \frac{1}{2}e^{-|x|} \quad (-\infty < x < \infty).$$

Ans. $E\xi = 0, D\xi = 2$.

14. Find the mean and variance of the random variable $\xi$ with probability density

$$p_\xi(x) = \begin{cases} \frac{1}{2b} & \text{if } |x - a| \leq b, \\ 0 & \text{otherwise.} \end{cases}$$

Ans. $E\xi = a, D\xi = b^2/3$.

15. The distribution function of a random variable $\xi$ is

$$\Phi_\xi(x) = \begin{cases} 0 & \text{if } x \leq -1, \\ a + b \arcsin x & \text{if } -1 \leq x \leq 1, \\ 1 & \text{if } x \geq 1. \end{cases}$$

Find $E\xi$ and $D\xi$.

Hint. First determine $a$ and $b$.

Ans. $E\xi = 0, D\xi = \frac{1}{2}$.

16. Let $\xi$ be the number of spots obtained in throwing an unbiased die. Find the mean and variance of $\xi$.

Ans. $E\xi = \frac{7}{2}, D\xi = \frac{3}{2}$.
