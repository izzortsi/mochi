Problem 9.16. Suppose $K : C \subseteq X \to C$ is a contraction and

$$x_{n+1} = K(x_n) + y_n, \quad \|y_n\| \leq \alpha_n + \beta_n \|x_n\|,$$

with $\lim_{n \to \infty} \alpha_n = \lim_{n \to \infty} \beta_n = 0$. Then $\lim_{n \to \infty} x_n = \overline{x}$.
