4. A random variable $\xi$ has probability density

$$p_{\xi}(x) = \frac{a}{x^2 + 1} \quad (-\infty < x < \infty).$$

Find

a) The constant $a$; b) The distribution function of $\xi$;
c) The probability $\mathbf{P}\{-1 \leq \xi \leq 1\}$.

Ans. a) $\frac{1}{\pi}$; b) $\frac{1}{2} + \frac{1}{\pi}$ arc tan $x$; c) $\frac{1}{2}$.

5. A random variable $\xi$ has probability density

$$p_{\xi}(x) = \begin{cases} ax^2 e^{-kx} & \text{if } 0 \leq x < \infty, \\ 0 & \text{otherwise} \end{cases}$$

$(k > 0)$. Find

a) The constant $a$; b) The distribution function of $\xi$;
c) The probability $\mathbf{P}\{0 \leq \xi \leq 1/k\}$.

6. A random variable $\xi$ has distribution function

$$\Phi_{\xi}(x) = a + b \arctan \frac{x}{2} \quad (-\infty < x < \infty).$$

Find

a) The constants $a$ and $b$; b) The probability density of $\xi$.

7. Two nonoverlapping circular disks of radius $r$ are painted on a circular table of radius $R$. A point is then “tossed at random” onto the table. What is the probability of the point falling in one of the disks?

Ans. $2(r/R)^2$.

8. What is the probability that two randomly chosen numbers between 0 and 1 will have a sum no greater than 1 and a product no greater than $\frac{2}{9}$?

Ans. $\frac{1}{3} + \frac{2}{9} \int_{1/3}^{2/3} \frac{dx}{x} = \frac{1}{3} + \frac{2}{9} \ln 2 \approx 0.49$.

9. Given two independent random variables $\xi_1$ and $\xi_2$, with probability densities

$$p_{\xi_1}(x) = \begin{cases} \frac{1}{2} e^{-x/2} & \text{if } x \geq 0, \\ 0 & \text{if } x < 0, \end{cases} \quad p_{\xi_2}(x) = \begin{cases} \frac{1}{3} e^{-x/3} & \text{if } x \geq 0, \\ 0 & \text{if } x < 0, \end{cases}$$

find the probability density of the random variable $\eta = \xi_1 + \xi_2$.

Ans. $p_{\eta}(x) = \begin{cases} e^{-x/3}(1 - e^{-x/6}) & \text{if } x \geq 0, \\ 0 & \text{if } x < 0. \end{cases}$
