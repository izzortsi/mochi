Nosso objetivo é, sabendo que os pontos $Q = (x,y,0)$ da cônica $c$ satisfazem a equação (em $x,y$)

$$Ax^2 + Bxy + Cy^2 + Dx + Ey + F = 0,$$

Suponhamos, pois, que o ponto $\bar{Q} = P_1 + u\vec{\epsilon}_1 + v\vec{\epsilon}_2$ corresponda ao ponto $Q = (x,y,0)$ de $c$. A reta $P_o\bar{Q}$ é dada por

$$P_o\bar{Q} = \{ P_o + t(x_1 - x_o, y_1 - y_o, z_1 - z_o) + tu(a_{11}, a_{21}, a_{31}) + tv(a_{12}, a_{22}, a_{32}), t \in \mathbb{R} \},$$

de forma que, fazendo com que a terceira coordenada se anule, obtemos

$$z_o + t(z_1 - z_o + a_{31}u + a_{32}v) = 0,$$

$$t = -\frac{z_o}{z_1 - z_o + a_{31}u + a_{32}v}.$$

Note que as contas acima vigoram apenas sob a hipótese $z_1 - z_o + a_{31}u + a_{32}v \neq 0$. O caso $z_1 - z_o + a_{31}u + a_{32}v = 0$ equivale a

$$z_1 + a_{31}u + a_{32}v = z_o,$$

que corresponde à interseção entre os planos $\beta$ e $\alpha_o = \{ (x,y,z) \mid z = z_o \}$ (note que $\alpha_o$ é o plano cego, paralelo a $\alpha$ e passando por $P_o$). Assim, $r$ pode ser: (i)uma reta, (ii)um plano, ou (iii)o conjunto vazio. No caso (ii), $\beta$ passa por $P_o$ e é paralelo ao plano $\alpha$, que contém $c$. Assim, $C \cap \beta = \{ P_o \}$.
