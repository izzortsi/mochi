VII) Se A é uma matriz ortogonal, det $A = \pm 1$. De fato, como A é ortogonal,

$$A^t A = I$$

logo,

det $(A^t A) = \det I$$

ou

$$(\det A^t) \times (\det A) = 1$$

mas,

det $A^t = \det A$,

portanto,

$$(\det A)^2 = 1$$

e

det $A = \pm 1$.

● Decorre dessa propriedade que todo operador ortogonal é inversível.
